Установка git и для чего это надо
Для чего нужен git?
Система git нужна для того, что-бы заливать свои сайты на хостинг ( грубо говоря в интернет ). А так же дает пользователям возможность работать вместе. Неважно где-бы вы не находились. Даже этот сайт сейчас работает на системе git. Github это сайт который позваляет загрузить ваш сайт в интернет. Так же это позваляет быстро посмотреть ваш сайт на телефоне или проверить адаптивность на Больших и маленьких мониторах.

Заходим в папку с index.html . Нам нужно запустить командную строку в этой папке. самый легкий способ это в корне папки вписать cmd и нажать Enter. Вот так:



Тык
Отлично, теперь вписываем: git init
Этим мы подключаем git к этой папке

Дальше, мы должны создать в git имя пользователя и емеил. Эту процедуру мы делаем 1 раз за все время.


git config --global user.name "тут имя на английском"
и
git config --global user.email email@email.com
нужно заметить то что имя мы пишем в "", а email без

Теперь, мы настроили git, пора заливать нашу папку в интернет.

git add -A
add - добавляет
-A - Все элементы
то-есть: git добавляет все элементы в так называемое хранилище.

Теперь это хранилище мы должны перевести в обновление (сommit).
git commit -a -m"имя commit"
commit - обновление
-a - все элементы
-m название
Но не забываем название писать слитно с -m
то-есть: -m"name"

все полностью готово к вылаживанию, но надо подключить github к git
